Ce projet est une introduction aux outils d'automatisations de type "Workflow".
Dans celui-ci nous exploiterons l'outils Knime qui nous permettra d'automatiser la création </br>
d'un rapport d'égalité hommes/femmes permettant de calculer ensuite un score, et ce de manière automatisée.

Les objectifs vont être :

- Se renseigner sur les méthodes de calcul du score d'égalité hommes/femmes en fonction de plusieurs paramètre
- Créer une partie de Workflow dédié à la préparation et à l'anonymisation des données
- Concevoir un workflow permettant de calculer des indicateurs en fonction des données actuelles ou de nouvelles données
- Calculer le score égalité hommes/femmes et proposer des axes d'améliorations
