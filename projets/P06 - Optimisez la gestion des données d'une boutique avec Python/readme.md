Ce projet nous permet d'entrée plus en profondeur dans l'analyse univariée avec Python pour <br />
permettre d'évoquer des axes d'amélioration sur le marketing et la gestion des données.

![alt text](https://github.com/nyann4/Portfolio-Repository/blob/main/projets/P6%20-%20Optimisez%20la%20gestion%20des%20donn%C3%A9es%20d'une%20boutique%20avec%20Python/Projet%206.png)


Les objectifs vont être :

- Comprendre le fonctionnement de L'ERP et les intéractions avec les différents fichiers
- Isoler les colonnes utiles pour l'analyse et les explorer
- Effectuer du nettoyage et des corrections sur les données
- Réaliser des analyses univariées approfondies
